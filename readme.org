#+TITLE: Utility Scripts and Tools Repository
#+AUTHOR: James Dyer

* Introduction

This repository contains a collection of utility scripts and tools for system management, media processing, desktop environment control, backups, and web publishing.

Below is a brief description of each script and its functionality.

** Shared Library

- *Common.sh*: Shared helper library sourced by other scripts. Parses file paths into components (extension, basename, directory, tag-stripped name, timestamp) used by tagging and conversion scripts.

** Media & Tagging Scripts

- *_tag_image_in.sh*: Adds tags to an image using the embedded metadata naming convention.
- *tag_image_out.sh*: Removes tags from an image filename and metadata.
- *tag_info.sh*: Displays metadata tags of media files.
- *tag_remove_meta.sh*: Strips all metadata from media files.
- *tag_time_correct.sh*: Corrects timestamp metadata in images or videos.
- *tag_video_out.sh*: Removes tags from a video file.
- *tagged_thumbnail_update.sh*: Regenerates missing thumbnail images for folder galleries; each folder should have a matching =.jpg= thumbnail.
- *tagged_validate_dir*: Validates that every subdirectory under the tagged content tree contains a required =index.md= file.
- *images.sh*: Processes a batch of image files.
- *images_output_cat.sh*: Outputs concatenated image results as files.
- *videos.sh*: Processes a batch of video files.
- *krita_create_videos.sh*: Creates video files from Krita animation projects.
- *photo-layout.sh*: Flexible photo print layout generator; arranges images at specified sizes onto a page for printing.
- *watermark.sh*: Resizes images and applies a watermark overlay.
- *update_all.sh*: Automated organisation and tagging of images and videos.
- *WhatsAppConvert*: Converts WhatsApp export date/time formats using the Common.sh naming convention.
- *ConvertNoSpace*: Renames files by replacing spaces in filenames.
- *find-duplicates.sh*: Finds duplicate files by comparing checksums.

** System & Utility Scripts

- *battery-monitor.sh*: Monitors the system battery status and sends desktop notifications.
- *cpu-speed.sh*: Displays the current CPU frequency.
- *led-monitor.sh*: Monitors and controls LED devices on the system.
- *disk-cleanup*: Interactive disk space cleanup for Arch Linux with btrfs/snapper support.
- *disk-initial-cleanup*: One-time setup script for disk space management (run as root).
- *disk-usage-report*: Displays a comprehensive disk usage breakdown.
- *nas-bypass*: Adds an explicit routing rule to keep NAS-subnet traffic off the Surfshark VPN tunnel, preventing NFS mount freezes.
- *startup.sh*: Runs programs and applies settings on desktop session startup.
- *startup_numpad.sh*: Starts the kmonad numpad keyboard configuration.
- *startup_root.sh*: Root-level startup script; handles NAS mounting and related setup.
- *off*: Quickly powers off or suspends the system.
- *upall.sh*: System update wrapper that calls the appropriate package manager (garuda-update or pacman).
- *mysync*: Personal file synchronisation script.
- *tidy_up_dir*: Cleans up and reorganises a directory.
- *download_video.sh*: Downloads videos from web sources via yt-dlp.
- *webpage_crawl.sh*: Crawls a webpage to download resources or content.

** Backup Scripts

- *do_backup*: Pushes the home set to the NAS via rsync (with image/video tidy-up).
  Its include/exclude lists now live in =bootstrap/lib/home-include.txt= and
  =bootstrap/lib/home-exclude.txt= — the shared single source of truth that also
  drives =bootstrap/sync-from-nas.sh= (pull) and =bootstrap/refresh-usb.sh= (stick).

** Screenshot & Screen Recording

- *screenshot.sh*: Takes a screenshot (auto-detects X11 or Wayland).
- *screenshot_wayland.sh*: Wayland screenshot using slurp for region selection and grim for capture.
- *screenshot_wayland_text.sh*: Takes a Wayland screenshot, runs OCR via tesseract, and copies the recognised text to the clipboard.
- *screen-record.sh*: Starts a screen recording session.
- *screen-record-check-status.sh*: Reports whether a screen recording is currently in progress.
- *top-and-tail.sh*: Trims a specified number of seconds from the start and end of a video file.

** Desktop & Wayland / Sway Scripts

- *launch_waybar*: Starts or restarts Waybar, the Wayland system monitoring bar.
- *toggle_waybar.sh*: Toggles Waybar visibility.
- *switch_waybar.sh*: Switches between different Waybar configurations.
- *toggle_rofi.sh*: Toggles the Rofi launcher; dismisses it if already open.
- *toggle_wlr_keyboard.sh*: Toggles the wvkbd on-screen virtual keyboard (Wayland).
- *toggle_x11_keyboard.sh*: Toggles the onboard on-screen virtual keyboard (X11).
- *keymap-load.sh*: Loads a predefined keymap configuration.
- *keymap-monitor.sh*: Monitors the current keymap state (e.g. sticky modifier status).
- *keymap-toggle.sh*: Toggles between sway and locked-modifier keymap configurations.
- *keyboard-reset.sh*: Resets the keyboard layout to defaults.
- *wallpaper_change.sh*: Changes the desktop wallpaper.
- *wallpaper_reject.sh*: Rejects the current wallpaper and restores the previous one.
- *terminal_pywal_tint.py*: Blends the pywal colour palette with each terminal's base theme to apply a gentle tint rather than a full palette override.
- *thumbnails-reset.sh*: Resets the thumbnail cache for images.
- *sway-print-tree.sh*: Prints the tree structure of the Sway window manager layout.

** QEMU & Virtualisation

- *qemu-manager-vm*: Standalone start/stop manager for QEMU VMs. Reads the same =~/VM/<name>/vm.conf= files as =qemu-manager.el= so VMs can be managed independently of Emacs.

** Web Publishing

- *web*: Website publishing script driven by =org-bootstrap-publish=. Handles building and deploying multiple sites with optional Cloudflare cache invalidation.
